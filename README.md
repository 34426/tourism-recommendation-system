# 旅游信息推荐系统


## 一、介绍

### springboot+mybatis+vue前后端分离的旅游信息推荐系统

开发语言：java

运行环境:idea或eclipse vscode 数据库:mysql

功能简介：系统分为前台跟后台两个项目

用户：  浏览旅游新闻、景区信息、美食信息、旅游线路、在线留言、旅游线路、订单信息

管理员： 账号管理、地区管理、景点信息管理、地方美食管理、旅游线路管理、订单信息管理、新闻管理、系统管理
